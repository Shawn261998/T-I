Binding: The Sifting Canvas

The goal of this project was to apply the skills I had developed across multiple web pages, creating my first website. This "book" included a homepage (its cover), navigation (table of contents), individual pages for each text (chapters), and my introduction (a colophon). It housed my texts and allowed a reader to move between them all.

Spreads: Graphic Design Bias

The goal of this project was twofold: achieving a successful responsive design and fostering effective collaboration between partners. Each duo sketched cooperatively and then implemented a joint execution together via pair programming. The final web page was responsive for mobile, desktop, and print layouts.

Manuscript: Design Education

The goal of this project was to refine basic skills in typography, focusing on expression, hierarchy, and appropriate form for a work. This was achieved through exploration, trial and error, and responding to critical feedback. Subsequently, this typesetting was executed in code, manifesting as a web page.
